# Renovate Configuration Presets

This repository contains shared Renovate configuration presets for commercetools and organization repositories.

## Usage

To use these presets in your repository, add the following to your `renovate.json`:

```json
{
  "extends": ["github>commercetools/renovate-config"]
}
```
